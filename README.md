<h1 id="argon-design-system"><a href="https://gogoboard.org/">GoGo Landing Page</a></h1>

<p>This project using <a href="https://www.creative-tim.com/product/vue-argon-design-system?__cf_chl_jschl_tk__=pmd_PstwgtdmfmVvt5m.HemX8PLDvXiRZzk.ygy_bvbP.Ck-1629271091-0-gqNtZGzNAjujcnBszQlR">Argon</a> which is a Vue js template.</p>

## Project requirements

This project using `Node.js` as an asynchronous event-driven JavaScript runtime and `npm` as a package manager. 
- node v12.x.x
- npm v6.x.x

## Get start
- Install all dependencies
  - npm i
- Run local host
  - npm run serve
